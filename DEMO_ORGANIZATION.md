## 🚀 使用方法

### 快速开始
```bash
# 进入demo目录
cd demo

# 查看所有示例
python load_examples.py

# 查看简单示例内容
python load_examples.py simple_bookmarks.txt

# 复制完整示例到剪贴板
python load_examples.py full_bookmarks.txt --copy
```

### 在主应用中使用
1. 运行主应用：`python pdf_bookmarker_gs.py`
2. 使用示例加载器复制示例内容
3. 在主应用中粘贴到目录文本区域
4. 点击"预览书签"检查结果
5. 生成带书签的PDF
